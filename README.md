# Simulation-Anomaly-Nuclear-Reactor

Python simulation of power measurements of a nuclear reactor with anomalies.

## Description 

This project simulate the power measurements of a nuclear reactor with anomalies by the use of Scikit-learn IsolationForest.
The simulation tracks all the measures and identify them as normal or anomaly.

## Requirements

- Python 3.x
- Numpy
- Matplotlib
- Scikit-learn
